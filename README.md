# Java Web Application using Embedded Tomcat

* This project is built using Oracle JDK 8, Html5 , JQuery 3.2.1 ,Javascript.

**How to use it ?**

```
- mvn clean install
- mvn package
- On Mac and Linux
-- sh target/bin/webapp
- On Windows
-- C:/> target/bin/webapp.bat
- Open a browser and go to http://localhost:8080/
```
